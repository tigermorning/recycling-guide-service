## ADDED Requirements

### Requirement: 사용자는 품목 이름으로 검색할 수 있다

시스템은 사용자가 입력한 품목 이름을 기반으로 분리수거 정보를 검색할 수 있는 기능을 제공하여야 한다.

#### Scenario: 정확한 품목명 검색

- **WHEN** 사용자가 "비닐"을 검색한다
- **THEN** 시스템은 "비닐"에 해당하는 분리수거 정보를 표시한다

#### Scenario: 부분 일치 검색

- **WHEN** 사용자가 "코팅"을 검색한다
- **THEN** 시스템은 "코팅된 종이" 등 "코팅"이 포함된 품목들의 분리수거 정보를 표시한다

#### Scenario: 검색 결과 없음

- **WHEN** 사용자가 존재하지 않는 품목을 검색한다
- **THEN** 시스템은 "검색 결과가 없습니다" 메시지를 표시한다

### Requirement: 검색은 실시간으로 수행된다

시스템은 사용자가 입력하는 즉시 검색 결과를 표시하여야 한다.

#### Scenario: 입력 즉시 검색

- **WHEN** 사용자가 품목 이름을 입력한다
- **THEN** 시스템은 입력이 완료된 후 즉시 검색 결과를 표시한다

### Requirement: 검색 결과에 세부 품목이 포함된다

시스템은 기본 카테고리뿐만 아니라 세부 품목도 검색 결과에 포함하여야 한다.

#### Scenario: 세부 품목 검색

- **WHEN** 사용자가 "충전선"을 검색한다
- **THEN** 시스템은 "충전선"의 분리수거 방법을 표시한다

#### Scenario: 혼합 소재 품목 검색

- **WHEN** 사용자가 "거울"을 검색한다
- **THEN** 시스템은 "거울(플라스틱+유리+건전지 혼합)"의 분리수거 방법을 표시한다
