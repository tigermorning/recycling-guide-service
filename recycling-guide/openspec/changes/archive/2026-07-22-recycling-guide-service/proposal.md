## Why

한국 사람들은 분리수거를 할 때 기본은 알지만 세부 기준을 모르는 경우가 많습니다. 코팅된 종이, 충전선, 혼합 소재 품목 등은 어떻게 배출해야 하는지 몰라 불편을 겪습니다. 이 문제를 해결하기 위한 품목별 분리수거 가이드 서비스를 만듭니다.

## What Changes

- **검색 기능**: 사용자가 품목 이름을 입력하면 분리수거 방법을 검색할 수 있는 기능
- **배출 방법 표시**: 검색 결과에 서울시 기준에 맞는 분리수거 방법을 한 화면에 표시
- **세부 품목 포함**: 코팅된 종이, 충전선, 혼합 소재 등 세부 품목도 검색 가능

## Capabilities

### New Capabilities

- `item-search`: 품목 이름으로 분리수거 정보를 검색하는 기능
- `disposal-guide`: 검색된 품목의 서울시 기준 분리수거 방법을 표시하는 기능

### Modified Capabilities

없음 (신규 프로젝트)

## Impact

- **프론트엔드**: 검색 입력 폼, 결과 표시 화면
- **데이터**: 서울시 분리수거 기준 데이터 (품목별 분리수거 방법)
- **의존성**: 없음 (순수 HTML/CSS/JS 또는 경량 프레임워크)
