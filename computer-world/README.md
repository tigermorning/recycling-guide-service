# How Computer Works - 컴퓨터가 어떻게 작동하는가

컴퓨터를 전혀 모르는 사람이 "컴퓨터가 실제로 어떻게 작동하는가"를 이해할 수 있는 인터랙티브 교과서입니다.

2.5D isometric 스타일의 3D 마을에서 건물을 클릭하며 컴퓨터의 내부 구조를 탐험합니다.

## Features

- **실세계** - 책상, 모니터, 컴퓨터 본체에서 시작
- **하드웨어 월드** - CPU, RAM, GPU, 저장장치 등 부품들의 물리적 구조
- **네트워크 월드** - DNS, 라우터, 방화벽, HTTP 규약의 동작 원리
- **내부 구조** - 각 부품의 내부를 확대해서 볼 수 있는 인테리어 모드
- **데이터 흐름 데모** - 로그인 버튼 클릭이 컴퓨터 내부를 통과하는 과정을 단계별로 시각화

## Built With

- [Three.js](https://threejs.org/) - 3D 그래픽 렌더링
- [Vite](https://vitejs.dev/) - 빌드 도구
- TypeScript

## Getting Started

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build
```

## How to Play

1. 브라우저에서 게임을 열면 인트로 화면이 표시됩니다
2. "마을 안으로 들어가기" 버튼을 클릭합니다
3. 실세계에서 **컴퓨터 본체**를 클릭하면 하드웨어 월드로 진입합니다
4. 각 건물을 클릭하면 컴퓨터 부품에 대한 상세 설명을 볼 수 있습니다
5. **데이터 흐름 데모** 버튼으로 전체 처리 과정을 자동으로 시청할 수 있습니다

### 조작법

| 조작 | 동작 |
|------|------|
| 왼쪽 드래그 | 맵 이동 |
| 오른쪽 드래그 | 시점 회전 |
| 스크롤 | 확대/축소 |
| 건물 클릭 | 설명 패널 표시 |
| 건물 더블클릭 | 내부 구조 보기 (인테리어 모드) |

## Learning Roadmap

```
실세계 (현재 위치)
    ↓
하드웨어 월드 (CPU, RAM, GPU, 저장장치)
    ↓
운영체제 월드 (커널, 스케줄러, 메모리 관리)
    ↓
브라우저 / 프론트엔드 월드 (HTML, CSS, JavaScript)
    ↓
자바스크립트 월드 (이벤트 루프, 비동기)
    ↓
백엔드 / 서버 월드
    ↓
데이터베이스 월드
    ↓
네트워크 월드 (DNS, 라우터, 방화벽, HTTP)
    ↓
보안 월드 / 배포 월드 / API 세계
```

## Project Structure

```
src/
├── core/           # 엔진, 카메라, 월드 관리
├── worlds/         # 실세계, 하드웨어, 네트워크 월드
├── buildings/      # 건물 컴포넌트
├── characters/     #NPC 캐릭터
├── interior/       # 부품 내부 구조 (인테리어 모드)
├── flow/           # 데이터 흐름 데모
├── portal/         # 월드 간 포털
├── ui/             # HUD, 패널, 인트로
├── learning/       # 학습 진행 추적
└── main.ts         # 게임 초기화
```
