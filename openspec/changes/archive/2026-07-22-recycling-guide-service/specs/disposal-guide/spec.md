## ADDED Requirements

### Requirement: 검색 결과에 분리수거 방법을 표시한다

시스템은 검색된 품목의 서울시 기준 분리수거 방법을 한 화면에 표시하여야 한다.

#### Scenario: 기본 분리수거 방법 표시

- **WHEN** 사용자가 "비닐"을 검색한다
- **THEN** 시스템은 다음 정보를 표시한다:
  - 분리수거함: 일반쓰레기
  - 배출 방법: 내용물 비우고 일반쓰레기 배출

#### Scenario: 세부 분리수거 방법 표시

- **WHEN** 사용자가 "코팅된 종이"를 검색한다
- **THEN** 시스템은 다음 정보를 표시한다:
  - 분리수거함: 일반쓰레기
  - 배출 방법: 코팅된 종이는 재활용 불가, 일반쓰레기 배출

#### Scenario: 혼합 소재 분리수거 방법 표시

- **WHEN** 사용자가 "거울"을 검색한다
- **THEN** 시스템은 다음 정보를 표시한다:
  - 분리수거함: 혼합 (부분별 분리 필요)
  - 배출 방법: 건전지 분리 후 건전지 분리수거함, 나머지는 일반쓰레기

### Requirement: 분리수거 방법은 서울시 기준을 따른다

시스템은 서울시 공식 분리수거 기준에 맞는 정보를 표시하여야 한다.

#### Scenario: 서울시 기준 정보 표시

- **WHEN** 사용자가任何 품목을 검색한다
- **THEN** 시스템은 서울시 기준에 맞는 분리수거 방법을 표시한다

### Requirement: 분리수거 방법은 직관적으로 표시된다

시스템은 사용자가 한눈에 분리수거 방법을 파악할 수 있도록 직관적으로 표시하여야 한다.

#### Scenario: 한 화면에 모든 정보 표시

- **WHEN** 사용자가 검색 결과를 본다
- **THEN** 시스템은 분리수거함, 배출 방법, 주의사항을 한 화면에 표시한다
