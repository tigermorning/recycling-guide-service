## Design

### UI 구조 변경

1. **검색 결과 영역 숨김 처리**
   - 기본 상태에서 `results` div에 `hidden` 클래스 적용
   - 검색 시에만 표시, 검색어 삭제 시 숨김

2. **컨테이너 너비 확장**
   - `max-width: 600px` → `max-width: 1100px`
   - 모바일 반응형 유지

3. **페이징 시스템**
   - 클래스: 페이지당 4개 (카드 높이 고려)
   - 빠른 참조: 페이지당 11개 (테이블 행 높이 고려)
   - `< Prev 1 2 3 Next >` 형태의 페이징 컨트롤

4. **빠른 참조 검색/필터**
   - 품목 검색 입력란
   - 배출 방법별 카테고리 드롭다운

### 데이터 확장

- 빠른 참조 항목 14개 → 47개로 확대
- PVC, PS, PP, HDPE, LDPE, OTHER 등 플라스틱 약자 추가
- 대형폐기물, 유해폐기물 등 카테고리 확대
- 거주지별 차이 안내 팁 포함
